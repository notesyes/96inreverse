96 in Reverse - Website Preview

Instructions

This page allows you to preview your 96 in Reverse podcast website. The preview below shows how your website will look when deployed.

Before deploying to GitHub and Coolify, make sure to:

  1. Add your logo image as images/logo.png
  2. Add host photos in the images/hosts/ directory
  3. Update host information in index.html
  4. Add a favicon using the favicon generator
  5. Make any other customizations you want

When you're ready to deploy, run the deploy.bat script to push your website to GitHub.
