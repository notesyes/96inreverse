96 in Reverse Podcast Website - Getting Started

1 Website Overview

Welcome to your 96 in Reverse podcast website! This website includes:

  • A modern, responsive design that works on all devices
  • Embedded RSS.com podcast player
  • Episode listings with descriptions
  • Host profiles section
  • About the show section
  • Social media links

The website uses the following color scheme:

  • #252a34 - Dark blue-gray (primary background)
  • #e2e8f2 - Light blue-gray
  • #eef1f7 - Very light blue-gray
  • #f5793b - Vibrant orange (accent color)

2 Adding Your Logo

To add your podcast logo:

  1. Prepare your logo image (recommended size: at least 500x500 pixels)
  2. Save it as logo.png in the images/ directory
  3. Make sure the image has a transparent background for best results

You can use the images/logo-placeholder.html file as a reference.

3 Adding Host Photos

To add photos for each host:

  1. Prepare square photos for each host (recommended size: at least 300x300 pixels)
  2. Save them in the images/hosts/ directory with the following names:
    • host1.jpg - First host
    • host2.jpg - Second host
    • host3.jpg - Third host
    • host4.jpg - Fourth host

You can use the placeholder-generator.html file to create temporary placeholder images if needed.

4 Updating Host Information

To update the information for each host:

  1. Open index.html in a text editor
  2. Locate the "Meet the Hosts" section (around line 100)
  3. For each host profile, update:
    • The host name (between <h3> and </h3> tags)
    • The host bio (between <p> and </p> tags)

Example:

<!-- HOST 1 -->
<div class="host-profile">
    <div class="host-image">
        <img src="images/hosts/host1.jpg" alt="Peter Smith">
    </div>
    <h3>Peter Smith</h3>
    <p>Peter is the founder and lead host of 96 in Reverse. With his background in journalism and passion for storytelling, he brings insightful perspectives to every episode.</p>
</div>

5 Creating a Favicon

A favicon is the small icon that appears in browser tabs and bookmarks for your website. To create a favicon:

  1. Open the favicon-generator.html file in your browser
  2. Click the "Download Favicon" button
  3. Save the file as favicon.ico in the root directory (same location as index.html)

Alternatively, you can create a custom favicon from your logo using an online favicon generator service.

6 Previewing Your Website

To preview how your website looks:

  1. Open the preview.html file in your browser
  2. This will show you a preview of your website in an iframe
  3. You can also click the "Open in New Tab" button to view the website in a full browser tab

Make any necessary adjustments to the content or styling before deploying.

7 Deploying to GitHub and Coolify

When you're ready to deploy your website:

  1. Make sure you have Git installed on your computer
  2. Run the deploy.bat script by double-clicking it
  3. This will initialize a Git repository, commit your files, and push them to GitHub
  4. Coolify will automatically detect the changes and deploy your website

If you encounter any issues during deployment, refer to the README.md file for troubleshooting steps.

8 Making Future Updates

To update your website in the future:

  1. Make your changes to the appropriate files
  2. Preview the changes using preview.html
  3. When satisfied, commit and push your changes to GitHub: 
    git add .
git commit -m "Description of your changes"
git push
  4. Coolify will automatically deploy the updated website
